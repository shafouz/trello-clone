# trello clone

## Install dependecies
```
yarn install
```

### Runs the project
```
yarn serve
```
server at localhost:8080

**Features**

- Multiple Boards
- Re-orderable lists and tasks
- Quick editing of lists and tasks
- Quick creation of lists and tasks

**Libraries and dependecies**
- Vuex, vuedraggable, vuex-persistedstate
- Jest for testing
- Lodash
- Bootstrap
